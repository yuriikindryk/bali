# CONTRIBUTE

**Developer Environment**

```bash
pip install -r requirements_dev.txt
``` 


**Tag a new release**

tag a version:

```bash
git tag -a v0.1.0
```

push tags to remote:

```bash
git push --tags
```


### Maintenance documents

```bash
# English version (default)
mkdocs serve
# Chinese version
cd docs/zh && mkdocs serve
```
