# config 

## configuration file

Configuration file is a python file location in project root directory.

```bash
todos
├── README.md
├── client.py
├── config.py   <-- configuration file
├── main.py
├── migrations
├── models.py
├── protos
├── resources.py
└── todo.sqlite
```

Todos Example project configuration file: [config.py](examples/todos/config.py)
